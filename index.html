What is CSS?

  1. CSS stands for cascading style sheets.

  2. CSS describes how HTML elements are to be displayed on screen, paper or in other media.

  3. CSS is not necessary
    css saves a lot of work.
    It can control layout of multiple pages all at once.

  4. External stylesheet are stored in css files.

CSS can be added to HTML documents in three ways:

  • Inline-By using style attribute inside html elements.
  • Internal- By using a style elemwnt in the head section.
  • External- By using a link element to link to an external css file.
  • The most common way to add CSS, is to keep the styles in external css files. However, in this tutorial we will use inline and Internal styles, Because it is easier to demonstrate, and easier for you to try it for yourself.

When comapring CSS class vs Id, the difference is that the css class applies the style to multiple elements. ID on the other hand, applies a style to one unique element. ID is also special in that you can use a special url to link directly to an element and it is used by javascript.

In css selectors are used to target a specific element or range of elements on a web page. Once an element has been targeted, a style or set of styles can be applied to the element.

There is a wide range of selectors available. Two of the most commonly used are class and id. Both are used to target an element to which style should be applied.
